# ポモドーロタイマー

シンプルなポモドーロタイマーのウェブアプリケーションです。

## 機能

- 作業時間（25分）
- 短い休憩（5分）
- 長い休憩（15分）
- タイマーコントロール（開始/一時停止/リセット）
- モダンなUIデザイン

## 使用方法

1. 「開始」ボタンをクリックしてタイマーを開始
2. 作業時間（25分）が終了すると自動的に短い休憩（5分）に切り替わります
3. 4回目の作業時間の後には長い休憩（15分）になります
4. 「一時停止」ボタンでタイマーを一時停止
5. 「リセット」ボタンでタイマーをリセット

## 技術スタック

- HTML5
- CSS3
- JavaScript
